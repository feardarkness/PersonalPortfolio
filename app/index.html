Acerca de mi:

Desarrollador Web ubicado en Bolivia, siempre aprendiendo y mejorando. Me encantan los retos, y mientras más complicado sea, mejor aún, la ideá de utilizar tecnología para resolver problemas es lo que más me apasiona.

Conocimientos en lenguajes de programación PHP, Python, Java y JavaScript; Bases de datos Oracle, MySql y PostgreSQL; Frameworks como Angular (Javascript), ADF (Java), Zend Framework (PHP) y Django (Python); maquetado de Sitios Web (HTML y CSS)

Experiencia en Desarrollo Ágil y con herramientas de versionamiento (Git, SVN).

Cualquier consulta, no dudes en contactarme, las críticas son bien recibidas.

Algunas páginas de prueba:

Reloj para administrar el tiempo con la técnica Pomodoro
Pomodoro es un método para administrar el tiempo creado por Francesco Cirillo. Se utiliza un reloj para dividir el trabajo en intervalos lo cuál te permite ser más productivo. Estos intervalos se denominan pomodoros.
Prueba de generación de mapa con leaflet y geojson
Prueba de la creación de un mapa con leafletjs, realizando la creación de los departamentos dinámicamente con datos geojson. Además cada departamento se ilumina cuando se pasa el mouse por encima.
Calculadora hecha solamente con css y javascript
Prueba de la creación de una calculadora solamente con javascript. Se pueden realizar las cuato operaciones básicas. La calculadora se creo para cumplir con los requerimientos de Freecodecamp.
Generador de frases celebres con la posibilidad de twitear la frase
Consumo de una API para generar frases celebres, además se agregó la funcionalidad que permite compartir por twitter la frase.
